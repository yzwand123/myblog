# myblog
javaSSM练手个人博客

## 所用技术
- mysql数据库
- SSM 后台
- java
- ZUI 前端UI框架

## 项目细节

- 数据库密码使用MD5加密
- Markdown 编辑器发布动态
- 图片上传至七牛云
- 内置聊天机器人
- 一周访客统计（图表显示）
